Premium To-Do List

Premium To-Do List is a task management web application built using HTML, CSS, and JavaScript. It helps users add, manage, and delete daily tasks through a clean and responsive interface.

Features

- Add new tasks
- Delete tasks
- Responsive design
- Simple and user-friendly interface

Technologies Used

- HTML5
- CSS3
- JavaScript
